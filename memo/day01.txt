- HTML이란?

- HTML(HyperText Markup Language)
	하나의 문서 안에서 보다 많은 정보를 구조화 할수있는 언어
	웹 문서를 만드는 언어

	웹 사이트의 화면을 구성하는 것들 중 뼈대의 역할을 한다.
	태그(Tag)를 사용하여 웹페이지가 어떤 구조로 만들어져 있는지 알려준다.
	태그란 해당 페이지에 관련된 여러 정보를 나타내는 구조물로 <>를 사용한다.

- HTML과 CSS
	웹개발에서 각각 웹 문서의 구조와 디자인을 담당하는 두가지 핵심언어

- HTML
	HTML은 웹페이지의 구조와 내용을 정의하는 마크업 언어
	(웹 문서의 뼈대 역할을 담당한다.)

- CSS(Cascading Style Sheet)
	CSS는 HTML문서의 시각적인 표현을 담당하는 언어
	(웹 문서의 디자인,스타일을 담당한다.)

- HTML 기본 구조
<html> : HTML문서의 시작을 나타내며 모든 HTML 요소를 포함한다.
	<head>
	헤드 태그 안에는 웹문서의 메타 정보를 넣어준다.
	웹문서의 뇌 역할을 한다.	
	<head>

	<body>
	본문 -> 실제 웹페이지에 나타나는 내용
	</body>
</html>

- html 기본 태그
<title> : 이 웹문서의 제목을 의미 (head안에서 사용한다.)
<h1> ~ <h6> : 제목 텍스트를 의미한다. (숫자가 작을수록 큰 제목을 나타낸다.)
<br> : 줄바꿈을 의미한다.

- Visual Studio Code 설치
구글 VsCode Download 검색 -> 최상단 링크 접속 -> 운영체제 맞는 버전 다운로드

- 태그(Tag)
	HTML에서 문서의 구조를 정의할때 사용되는 문법
	
	<h1> 안녕하세요 </h1>

1. 여는 태그 : 태그의 이름과 열고닫는 꺽쇠로 구성,
	     태그의 시작을 의미한다.

2. 닫는 태그 : 태그의 이름앞에 슬래시(/)로 구성,
	     태그의 종료를 의미한다.

3. 내용(Content) : 요소의 내용
4. 요소(Element) : 여는태그,요소,닫는 태그를 통틀어 하나의 요소라고 부른다.

- 태그의 관계
<부모의 여는태그>
	<자식의 여는태그> </닫는태그>
	<자식의 여는태그> </닫는태그>	
</부모의 닫는태그>

** 자식태그들끼리 동일선상에 있으면 형제태그라고 부른다.

- 주석(Comment)
	코드에 관한 설명을 작성하는 공간
	사람간의 소통을 위해 작성

	** 프로그램 실행에 전혀 영향을 미치지 않는다!

	<!-- 주석 처리할 문장 -->
	단축키 : ctrl + /

- HTML 기본구조 2
	** html 파일을 만들고 ! + enter를 누르면 기본구조를 자동으로 만들어준다.

<!DOCTYPE html> : html문서의 형식이 HTML임을 나타낸다.

<meta charset="utf-8"/> : HTML문서의 문자 인코딩 형식을 UTF-8 로 지정하겠다.
			  한글문자도 호환하겠다.

<meta name= "viewport" ,..../>
	뷰포트는 기기에 따라서 어떻게 화면이 처음에 보일지를 설정하는 속성값
	(CSS파트에서 알아보자)

- 확장 프로그램
1. Live Server : html 웹문서를 실시간으로 확인할수 있는 확장프로그램
	        (단축키 : alt + L + O)

2. prettier : VsCode코드 자동으로 태그를 정렬해주는 확장프로그램

3. Auto Rename Tag : 닫는태그를 자동으로 지정해주는 확장프로그램

- 기본 태그의 종류
<p> </p> : 단락태그, 단락을 위/아래 간격이 벌어지도록 자동으로 줄바꿈 해준다.
<strong></strong>, <b></b> : 굵은 텍스트
<hr> : 가로선을 그어주는 태그 , 닫는 태그가 없다. => 빈 태그

- 이외의 기본태그
<i></i> : 태그안에 들어온 텍스트를 기울여서 표시한다.
<mark></mark> : 태그안에 들어온 텍스트를 형광펜으로 칠한것처럼 강조한다.
<ins> </ins> : 태그안에 들어온 텍스트 아래에 선을 표시한다.
<del> </del> : 태그안에 들어온 텍스트의 중간에 선을 표시한다.
<sub></sub> : 아랫첨자 텍스트를 표시한다.
<em> </em> : 강조된 텍스트를 표현할 때 사용한다.
<marquee> </marquee> : 태그안에 들어온 텍스트를 좌우상하로 움직인다. 

- 속성(Attribute)
	1. 닫는 태그가 아닌 여는 태그에 작성된다.
	2. 태그의 이름뒤에 작성한다.
	3. 주로 속성명 = "속성값" 구조로 이루어져 있다.
	ex) <marquee direction = "right"> content </marquee>
	4. 태그는 각각에 사용될수 있는 여러 속성을 가지고 있다.
	5. 하나의 태그에 여러개의 속성을 지정할 수 있다. 이 때 각 속성은
	   띄어쓰기로 구분하여 작성한다.

	6. 몇몇 속성은 속성값 없이도 사용될 수 있다. 
	   이러한 경우는 속성명만 작성해준다.

- 리스트 태그(List Tag)
목록 태그라고도 부르며 텍스트나 이미지같은 내용을 목록화하여 정리하며
카테고리 등등을 표현하기 위해서 사용한다.

- 리스트 태그의 종류
1. <ol></ol> : 
	Ordered List 약자로 순서가 있는 목록을 만든다.
	
2. <ul></ul>
	UnOrdered List 약자로 순서가 없는 목록을 만든다.

3. <li> </li>
	list의 약자, <ol> 또는 <ul>의 자식태그로 사용한다.

- 리스트 태그의 구조
<ol>
	<li> 내용1 </li>
	<li> 내용2 </li>
</ol>

<ul>
	<li>내용1</li>
	<li>내용2</li>
</ul>

- <ol> 태그의 속성
1. start = "시작값" : 순서의 시작값을 지정할 수 있다.

2. type = "A,a,I,1"
	"A" : 알파벳 A부터 순서를 지정한다(대소문자)

	"I" : 로마 숫자로 순서를 지정한다.(대소문자)

	"1" : 숫자로 순서를 지정한다.(type속성을 지정해주지 않으면 기본값)

- <ul> 태그의 속성
1. type = "속성값"
	circle, disc(기본값), square , none(자주 사용)

- HTML 엔티티(Entity)
	HTML에서 이미 역할(의미)이 부여되있는 문자	
	HTML엔티티를 사용하여 다양한 특수문자,텍스트,이모지를 표현할수 있다.

- 엔티티 사용 문법
&엔티티이름;

- HTML 엔티티 종류
1. &lt; : <
2. &gt; : >
3. &amp; : &
4. &quot; : "
5. &apos; : '
