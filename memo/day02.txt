- HTML과 CSS
	웹개발에서 각각 웹 문서의 구조와 디자인을 담당하는 두가지 핵심언어

- HTML(HyperText Markup Language)
	HTML은 웹페이지의 구조와 내용을 정의하는 마크업언어
	웹문서의 뼈대 역할을 담당한다.

- CSS(Cascading Style Sheet)
	CSS는 HTML문서의 시각적인 표현을 담당하는 디자인 언어
	웹문서의 디자인이나 스타일링을 담당한다.

- 태그(Tag)
	HTML에서 문서의 구조를 정의할때는 사용되는 문법

	1. 여는태그
		태그의 시작을 알리는 태그		

	2. 닫는태그
		태그의 끝을 알리는 태그

	3. 빈태그
		닫는태그가 없는 태그

	4. 내용 
		요소의 내용

	5. 요소
		여는태그,내용,닫는태그를 통틀어 하나의 요소라고한다.

- 기본 태그의 종류
1. <title> : 웹 문서의 제목을 의미한다.
2. <h1> ~ <h6> : 제목 텍스트
3. <br> : 줄바꿈
4. <p> : 단락(문단) 태그
5. <strong> , <b> : 굵은 텍스트
6. <hr> : 가로선을 그어주는 태그
7. <i> : 기울임 텍스트
8. <mark> : 색칠 강조 텍스트
9. <ins> : 텍스트 아래에 선을 그어준다.
10. <del> : 텍스트 중간에 선을 그어준다.
11. <sub> : 아랫첨자 텍스트
12. <em> : 강조 텍스트(기울임)
13. <marquee> : 태그안에 텍스트를 상하좌우로 움직인다.

- 태그의 속성(Attribute)
1. 여는 태그의 이름뒤에 작성한다.
2. 주로 속성명 = "속성값"	
3. 하나의 태그에 여러 속성을 지정할 수 있고 이때 각 속성은 띄어쓰기를 구분하여 사용한다.
<ul type="" start = "">

4. 몇몇 속성은 속성값없이도 사용이 가능하다. 이러한 경우는 속성명만 작성해준다.

- 리스트 태그(List Tag)
	목록태그라고 부르며 텍스트나 이미지같은 내용을 목록화여 정리할때 사용한다.
	카테고리를 표현할때 주로 사용한다.

- 리스트 종류
1. <ol></ol> : 순서가 있는 목록을 만든다.

2. <ul></ul> : 순서가 없는 목록을 만든다.

3. <li> </li> : 실질적은 리스트 내용을 넣어준다.

- HTML 엔티티
	HTML에서 이미 역할이 부여되어있는 문자
	엔티티를 사용하여 다양한 특수문자, 텍스트, 이모지를 표현할수 있다.

	&엔티티이름;

- 테이블 태그(Table Tag)
행과 열의 격자 형태로 내용을 나타내는 태그
주로 데이터를 정렬화하고 그룹화할때 사용한다.

- 테이블 태그 종류
<table> </table> : 표를 만드는 태그
	1. 테이블 태그는 표를 만들때 사용하며 표의 제목, 행의 수,열의 수 등등은
	   다른 태그를 이용해 표현한다.

<caption> </caption>
	표의 제목을 의미하는 태그

<tr> </tr>
	행을 만드는 태그

<td> </td>
	열을 만드는 태그

<th> </th>
	제목행에 컬럼을 만드는 태그

- table태그의 속성
border : 테이블의 테두리 선 굵기를 의미한다.
width : 테이블의 가로크기를 의미한다.
height : 테이블의 세로크기를 의미한다.
cellspacing : 테이블 셀간의 간격을 의미한다.

- td태그의 속성
colspan : 열 셀 합치기 , 속성값으로 몇개의 열을 차지할건지 지정
rowspan : 행 셀 합치기 , 속성값으로 몇개의 행을 차지할건지 지정

- 이미지 삽입 태그

- 파일의 경로
1. 파일이 존재하는 위치(디렉토리,폴더의 위치)
2. 파일이 존재하는 경로를 찾아가기 위한 방법은 상대경로와 절대경로가 있다.

- 절대 경로
	파일이 존재하는 최상위 디렉토리부터 추적하여 파일이 존재하는 정확한 위치를 표시하는 경로

	파일의 정확한 위치를 표시하기 때문에 어느곳에서든 파일의 주소만 알면 접근이 가능하다.
	단 파일이 존재하는 절대경로가 변경되면 일일히 수정해줘야한다.
	대부분의 브라우저에는 보안상의 문제로 절대경로를 통해서 로컬이미지를 가져오는것을 막아놓는
	경우가 있다.

	ex) D드라이브 => HTML폴더 => image폴더 => aa.jpg파일

	=> D:\HTML\image\aa.jpg ** 디렉토리의 안을 의미할 때는 역슬래시(\)를 사용한다.

- 상대경로
	1. 현재 작업하고 있는 파일을 기준으로 가져오려는 파일의 상대적인 위치를 표시하는 경로
	2. 작업중인 파일의 위치를 다른 폴더로 이동시킬때 더 쉽게 관리할 수 있다.

../ : 현재 폴더 기준 상위 경로(나를 가지고 있는 디렉토리)
./ : 현재 폴더가 존재하는 현재 경로(주로 생략)
/ : 루트 경로 : 작업 공간

- 이미지 삽입 태그
<img src="이미지 파일 경로" alt="이미지 설명"/>
src 속성은 넣고싶은 이미지 경로를 작성하며 , 웹상에 존재하는 이미지 경로를 가져와서 입력해도 된다.
alt 속성은 이미지에 설명을 작성하며 만약 어떠한 이유로 이미지가 깨지는 경우 설명이 대체되어 나온다

- a 태그(앵커 태그)
	다른 링크로 이동할수 있게 해주는 태그

<a href="링크주소나 경로"> 링크를 걸어줄 내용 </a>
a태그로 감싼 내용은 클릭 시 해당 경로의 링크로 이동한다.

- a 태그의 target 속성
	링크를 열 위치를 지정한다.

	_blank : 새 창 또는 새 탭에서 링크를 열어준다.

	_self : 현재 창에서 링크를 열어준다 (기본값)

- 인라인,블록요소
	HTML의 모든 요소는 인라인속성과 블록속성으로 나뉜다.
	
- 인라인 요소(Inline Content)
	1. 내용물의 크기만큼만 영역을 차지하는 요소
	2. 내용물의 크기만큼만 영역을 차지하므로 다른 요소와 같은줄에 위치할 수 있다.
	   즉, 여러 인라인 요소를 사용했을 때 줄바꿈되지 않는다.
	3. 대표적으로 span, img,strong,i 태그 등등이 있다.

- 블록 요소(Block Content)
	1. 내용물과 상관없이 한줄을 모두 차지하는 요소
	2. 한줄을 모두 차지하므로 다른요소와 같은줄에 위치할 수 없다.
	   즉 가로너비 속성이 100%를 모두 차지하고 줄바꿈이 된다.
	3. 대표적으로 h1, div, p태그 등등이 있다.
	4. 다른 여러 요소들을 하나의 블럭으로 묶어주기 위한 용도로도 사용된다.

- div태그와 span태그
	<span> </span> : 대표적인 인라인 태그
	<div> </div> : 대표적인 블록태그

1. span태그와 div태그 둘다 의미가 없는 태그(시멘틱하지 않은 태그)
2. 각각 인라인요소와 블록요소이므로 주로 여러 태그를 묶어서 화면의 레이아웃을 나누는 용도로
   사용된다.
3. HTML의 구조(레이아웃)를 잡는 굉장히 중요한 태그들이다.

- CSS (Cascading Style Sheet)
	1. HTML로 만든 웹 문서의 스타일(디자인)을 담당하는 언어
	2. 스타일이란 글꼴,색상,테두리,정렬,배치 방법 등 외관을 의미한다.
	3. HTML에서 제공하는 속성으로도 간단한 스타일을 지정할 수 있다.
	   그러나 내용과 디자인이 분리되어있어야 유지보수에 유리하므로 CSS를 사용하는게 좋다.

- CSS 형식
	속성명 : 속성값;
	속성명2 : 속성값;
	,...

	속성과 속성값은 미리 정해져 있으며 필요에따라 검색하여 사용한다.

- CSS 적용방법
	작성한 HTML 태그에 CSS를 적용하는 방법은 3가지가 있다.
	인라인 스타일 , 내부 스타일, 외부 스타일

1. 인라인 스타일
	스타일을 적용시키고 싶은 태그에 style 속성을 사용하여 CSS속성값을 지정해줄수 있다.
	<태그명 style=" CSS속성명 : 속성값; ">
	** 제일 권장하지 않는 방법 , 정말 급할때만 사용

- 기본 CSS 속성
background-color : 요소의 배경색을 지정한다.
color : 텍스트의 색상을 지정한다.
font-size : 텍스트의 크기를 지정한다.(px단위)
font-weight : 텍스트의 굵기를 지정한다.
	normal : 기본 텍스트굵기
	bold : 두껍게 표시된 텍스트
	bolder : 부모요소의 텍스트크기보다 더 두껍게 표시된 텍스트
	lighter : 부모요소의 텍스트 크기보다 더 얇게 표시된 텍스트
	100 ~ 900 : 100이 가장 얇은 텍스트, 900이 가장 두꺼운 텍스트

- 내부 스타일
	<head>태그 영역에 <style></style> 태그를 이용하여 각 요소들의 스타일을 정의한다.
