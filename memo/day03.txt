- 테이블 태그(Table Tag)
   행과 열의 격자 형태로 데이터를 나타내는 태그
   주로 데이터를 정렬화하고 그룹화할 때 사용한다.

- 테이블 태그 종류
1. <table></table> : 표를 만드는 태그
   테이블 태그는 표를 만들 때 사용하며 표의 제목, 행의 수, 열의수 등등은
   다른 태그를 이용해 사용한다.

2. <caption> : 표의 제목을 의미하는 태그

3. <tr> : 행을 만드는 태그

4. <td> : 열을 만드는 태그

5. <th> : 제목행에 컬럼을 만드는 태그

- table 태그의 속성
   1. border : 테이블의 테두리 선 굵기를 지정
   2. width : 테이블의 가로크기
   3. height : 테이블의 높이를 지정
   4. cellspacing : 테이블 셀간의 간격을 의미한다.
   
- td 태그의 속성
   colspan : 열 셀 합치기, 속성값으로 몇개의 열을 합칠건지 지정
   rowspan : 행 셀 합치기 , 속성값으로 몇개의 행을 합칠건지 지정

- 이미지 삽입 태그
   
- 파일의 경로
   1. 파일이 존재하는 위치(디렉토리의 위치)
   2. 상대경로와 절대경로 방식이 있다.

- 절대경로
   파일이 존재하는 최상위 디렉토리부터 추적하여 파일이 존재하는 정확한 위치를 표시하는것.

   파일의 정확한 위치를 표시하기 때문에 어느곳에서든 주소만 알면 접근이 가능하다.
   대부분의 브라우저에서 보안상의 문제로 절대경로를 통해서 로컬이미지를 가져오는 것을 막아놨다.

   
- 상대경로
   1. 현재 작업하고있는 파일을 기준으로 가져오려는 파일의 상대적인 위치를 표시하는 방법
   2. 작업중인 파일의 위치를 다른 폴더로 이동시킬 때 유연하게 관리할 수 있다.

../ : 현재 폴더를 기준으로 상위 경로(나를 가지고 있는 디렉토리)
./ : 현재 폴더가 존재하는 현재 경로(주로 생략)
/ : 루트 경로 => 작업공간 폴더

- 이미지 삽입 태그
<img src="이미지 파일의 경로" alt="이미지 설명"/>
src속성은 불러오고싶은 이미지의 경로를 작성하며, 웹상에 존재하는 이미지 주소를 가져와서 작성할 수 있다

alt속성은 이미지의 설명을 작성한다. 어떠한 이유로 이미지가 깨졌을 때 alt속성값이 대체되어서 나온다.

- a 태그(앵커 태그)
   다른 링크로 이동할수 있게 해주는 태그

<a href="링크주소나 경로"> 링크를 걸어줄 컨텐츠 </a>
a 태그로 감싹 컨텐츠는 클릭 시 해당 경로의 링크로 이동한다.

** 단순 페이지 이동만 할때 사용한다.

- a 태그의 target 속성
   링크를 열 위치를 지정한다.

   _blank : 새 창 또는 새 탭에서 링크를 열어준다.
   
   _self : 현재 창에서 링크를 열어준다(기본값)

- 인라인, 블록요소
   HTML의 모든 요소는 인라인속성 , 블록속성으로 나뉜다.
   
- 인라인 요소(Inline Content)
   1. 내용물의 크기만큼만 영역을 차지하는 요소
   2. 내용물의 크기만큼만 영역을 차지하므로 다른 요소와 같은 줄에 위치할 수 있다.
   3. 대표적으로 span, img, strong,.... 등등이 있다.

- 블록 요소(Block Content)
   1. 내용물과 상관없이 한줄을 전부 차지하는 요소
   2. 한줄을 전부 차지하므로 다른 요소와 같은 줄에 위치할 수 없다.
   3. 대표적으로 h1, div, p태그 등등이 있다.
   4. 다른 여러 요소들을 하나의 블럭으로 묶어주기 위한 용도로 사용한다.

- div태그와 span태그
   <span> </span> : 대표적인 인라인 태그
   <div> </div> : 대표적인 블록 태그

   1. 둘다 의미가 없는 태그(시맨틱하지 않은 태그)
   2. 각각 인라인 요소와 블록요소이므로 주로 여러태그를 묶어서 화면의 레이아웃을 나누는 용도로
      사용한다.
   3. HTML의 구조를 잡는 중요한 태그다.

- CSS (Cascading Style Sheet)
   1. HTML로 만든 웹문서의 디자인을 담당하는 언어
   2. HTML에서 제공하는 속성만으로도 간단한 디자인을 입힐수 있지만
      내용과 디자인이 분리되어있어야 유지보수에 유리하므로 css를 사용하는것이 좋다.

- CSS 형식
   속성명 : 속성값;

   ** css속성과 속성값은 미리 정해져 있으며 필요에 따라 검색하여 사용한다.

- CSS 적용방법
1. 인라인 스타일
   스타일을 적용시키고 싶은 태그에 style 속성을 사용하여 css속성값을 지정할 수 있다.
   <p style=" css속성명 : 속성값;" > </p>
   ** 제일 권장하지 않는 방법

2. 내부 스타일
   <head>영역에 <style> 태그를 사용하여 각 요소들의 스타일을 정의해줄수 있다.

- 형식
<head>
   <style>
   선택자{
      속성명 : 속성값;
      속성명2 : 속성값2;
      ,....
   }

   </style>
</head>

선택자(selector) : css에서 html의 어떤 요소에 스타일을 입힐지 선택하기 위해 사용되는것

- id속성과 class속성
   id와 class속성을 사용하여 특정 태그들을 식별하고 그룹화 할 수 있다.
   HTML의 모든 태그는 id,class속성을 사용할 수 있다.

- id속성
   특정 태그에 식별가능한 고유한 값을 지정한다.
   CSS에서 선택자로 사용할 때는 #id명 으로 사용한다.

   ** 각기 다른 태그에 중복되는 id값을 설정해도 오류는 나지 않는다.
   단, id속성의 사용목적에 맞지 않는다.(id속성값은 고유해야한다.)
   태그를 그룹화하는 목적으로는 class속성을 사용한다.

- class 속성
   여러 태그를 하나의 집단으로 묶어준다.
   <태그명 class="클래스명">
   css에서 선택자로 사용할 때는 .class명으로 선택해준다.

   ** 하나의 태그가 id와 class속성을 동시에 가질 수 있으며 class를 여러개 가질수도 있다.
   css를 입힐때는 주로 식별자로 class를 많이 사용하며
   id속성은 그 웹페이지에서 중요하고 유일한 요소에 부여하는것을 권장한다.

- 외부 스타일(가장 많이 사용)
   별도의 css파일을 만들어 css를 적용한다.
   css와 html파일을 완전 분리하여 관리할 수 있기 때문에 가독성과 유지보수에 유리하다.
   
   css파일을 html파일에 적용하기 위해서는 <head>영역안에 <link>태그를 사용한다.

- CSS파일 연동 방법
<head>
   <link rel="stylesheet" href="css파일의 경로"/>
</head>

rel속성은 현재 html파일과 연동할 파일의 관계를 의미하며 href는 적용시킬 css파일의 경로를 작성한다.

- 박스 모델(Box Model)
HTML의 모든 요소는 일정한 영역을 차지하고 직사각형의 모양으로 차지한다.
css에서는 이런 영역들을 Box Model

- 박스 모델의 구성
1. margin : 바깥 여백(테두리 밖의 여백)

2. border : 컨텐츠의 테두리

3. padding : 안쪽 여백(테두리와 컨텐츠 사이 여백)

4. contents : 컨텐츠의 영역

** 인라인 요소의 박스 모델도 margin,border,padding 설정이 가능하다.
   단, width속성이나 height속성으로 컨텐츠의 크기를 설정할 수 없다.

- 박스모델의 css속성
1. contents
   width  : 가로 너비
   height : 세로 높이

   1. 단위는 기본적으로 px단위이며 그 외에도 여러가지 단위가 있다.
   2. 인라인 요소는 불가능하다.

2. border
   border : 굵기 형태 색상
   
   - 형태 속성값
   solid : 직선
   dotted : 점선
   dashed : 긴 점선

3. padding
   padding : 4방향;
   padding : 상하 좌우;
   padding : 상 좌우 하;
   padding : 상 우 하 좌;

4. margin
   margin : 4방향;
   margin : 상하 좌우;
   margin : 상 좌우 하;
   margin : 상 우 하 좌;
   
   ** auto속성값을 사용하면 요소를 중앙에 위치하게 만든다.(부모 요소 크기 기준)
   ** 블록요소를 여러개 배치하게 되면 각 margin값이 겹쳐지게 된다.

- display 속성
display속성은 HTML에서 요소를 어떻게 표시할지 결정한다.
HTML의 모든 요소들은 기본적으로 display속성값을 가지고 있다.

display : block => 요소를 블록요소로 표현
display : inline => 요소를 인라인 요소로 표현하겠다.

- display속성의 기본 4가지값
1. block : 요소의 속성을 블록요소로 설정한다.
      블록요소는 기본적으로 가로영역을 모두 차지한다.

2. inline : 요소의 속성을 인라인요소로 설정한다.
       인라인 요소는 기본적으로 컨텐츠의 크기만큼만 영역을 차지한다.
       블록요소와 달리 컨텐츠의 가로,세로 크기를 지정할 수 없다.

3. inline-block : 요소의 속성을 인라인 블록으로 설정한다.
      인라인 블록은 인라인속성과 블록속성의 특징을 섞어놓은 속성이다.
      컨텐츠 크기만큼만 영역을 차지하는데 가로,세로 크기를 지정할 수 있다.
      내부적으로는 block요소 외부적으로는 inline요소의 특징을 갖는 속성

4. none : 요소를 보이지 않게 설정한다.
     단순히 요소를 투명하게 만드는게 아니라 아예 사라지게 만든다.
     공간도 차지하지 않는다.

- display 실습
스타일 태그를 이용해 내부 스타일로 작성

1. inline 텍스트를 가진 span요소를 3개 생성
   배경색 : 노란색
   테두리 : 1px 빨간색 실선
   요소 사이의 간격 : 5px
   텍스트와 줄사이의 간격(line-height) : 10px

2. inline block텍스트를 가진 div요소를 3개 생성
   배경색 : 초록색
   테두리 : 2px 파란색 점선
   요소 사이의 간격 : 20px
   가로 : 100px , 높이 : 100px
   속성을 인라인블록으로 지정
